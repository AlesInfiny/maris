# ローカル開発環境の構築 {#top}

SSR アプリケーションの開発におけるシステム要件およびローカル開発環境の構築方法は、 原則 CSR 編と同一です。
そのため、 [CSR編 > ローカル開発環境の構築](../csr/local-environment/index.md) を参照し、必要なソフトウェアをインストールしてください。

ただし、 npm パッケージの利用を必須としないので、 [ローカル開発環境の構築 > Node.js のインストール](../csr/local-environment/index.md#install-nodejs) は実施不要です。

また、フロントエンドの開発フレームワークとして Blazor を用いるため、 CSR アプリケーションの開発時と比較して、より Visual Studio の利用を推奨します。
