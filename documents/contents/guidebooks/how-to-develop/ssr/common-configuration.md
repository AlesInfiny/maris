# プロジェクトの共通設定 {#top}

C# プロジェクトファイル（ csproj ）に対して共通して行うべき設定は、 アプリケーションの形態が CSR であるか SSR によって変わりません。
そのため、 [CSR編 > プロジェクトの共通設定](../csr/dotnet/project-settings.md) に従い、下記の設定を実施します。

- .NET SDK の設定
- プロジェクトファイルの設定
- パッケージ集中管理の導入
- 静的コード解析用パッケージと設定ファイルの導入
- メッセージリソースの追加
